# Smartphone Shop

Some features:
* Authentication & registration, Firebase authentication
* Multiple screens via StackNavigator (React navigation)
* Settings/Preferences
* List of Card views
* Input data via forms
* Toolbar
* ViewPager / ScrollableTabs
* Firebase Realtime Database
* DrawerNavigator
* TabNavigator
* YouTube
* Camera
* Gallery
* Map
* Animations
